About This Project

University-level Operating Systems Project

📚 Project Overview

This is a comprehensive thread management simulator demonstrating CPU scheduling algorithms at a professional level. It combines real-time visualization with accurate algorithmic implementation.

🧠 Algorithms Implemented

  • FCFS: First Come First Serve
  • SJF: Shortest Job First
  • Priority: Non-preemptive Priority Scheduling
  • Round Robin: Time quantum-based scheduling

⚙️ Technical Stack

  • Frontend: HTML5, CSS3, Vanilla JavaScript
  • Backend: Node.js + Express
  • Design: Glassmorphism + Modern UI/UX
  • Features: Real-time animation, comparison mode

📊 Metrics Calculated

  • Waiting Time: Time thread waits for CPU
  • Turnaround Time: Completion time - Arrival time
  • Response Time: First execution - Arrival time

🎯 Test Case Example

T1: AT=0, BT=5, Priority=1
T2: AT=1, BT=3, Priority=2
T3: AT=2, BT=1, Priority=3

FCFS: T1(0-5) → T2(5-8) → T3(8-9)
SJF:  T1(0-5) → T3(5-6) → T2(6-9)
Priority: T1(0-5) → T2(5-8) → T3(8-9)

🚀 Features

  • ✅ Add/Edit/Delete threads
  • ✅ Bulk random generation
  • ✅ Real-time animation
  • ✅ Algorithm comparison
  • ✅ Handles 100+ threads
  • ✅ Professional dashboard UI
